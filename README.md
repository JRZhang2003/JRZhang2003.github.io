# Personal Homepage (Pure Static HTML)

This project has been simplified to a pure static website:

- No Jekyll
- No build step
- No template inheritance

## Main Structure

- `index.html`
- `_Publications/`
- `_Talks/`
- `_Portfolio/`
- `_BlogPosts/`
- `_CAU/`
- `_PKU/`
- `_CV/`
- `assets/` (`css`, `js`, `images`)
- `files/` (`publications`, `CAU`, `PKU`)

## Maintenance

1. Edit `index.html` directly inside each module folder.
2. Put PDFs into `files/publications`, `files/CAU`, or `files/PKU`.
3. Open `index.html` in a browser, or serve this folder with any static server.
